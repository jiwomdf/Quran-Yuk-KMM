# Quran-Yuk-KMM

<b>Quran Yuk</b> is an Al-Qur'an <b>Kotlin Multiplatform App</b>, it has online and offline support with boundresource, use clean code architecture principles, and also unit tested for both Android and iOS  üåô<br><br>

This app consume the https://api.alquran.cloud/ Endpoint
